Le Trot - Cheval Français

Réécriture de l'application d'information sur les courses hippiques pour Le Trot, l'autorité française des courses de trot attelé.

Contributions Clés

  • Réécriture Complète : Réécrit le code source pour les versions iPhone et iPad
  • Compatibilité iOS 5+ : Rendu l'application compatible avec iOS 5 et les versions ultérieures
  • Notifications Push : Corrigé les problèmes critiques de notifications push

Technologies Utilisées

Objective-C, Apple Push Notification Service, iOS SDK

Impact

Fourni aux passionnés français de courses de trot attelé des informations fiables et en temps réel sur les courses, résultats et actualités.
