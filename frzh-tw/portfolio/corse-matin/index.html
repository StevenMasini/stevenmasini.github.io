Corse Matin per noï

Corse Matin報社的數位報紙閱讀器，提供日報和週報的增強PDF檢視功能。

主要功能

  • MiLibris API整合：整合MiLibris API的v3和v4版本以實現增強PDF檢視
  • iPad版本：設計並開發了iPad最佳化版本
  • 長期相容性：為iOS 4.3+構建，在iOS 11上仍能執行而無需重大更新

技術成就

創建了一個強健、經得起未來考驗的應用程式，在多個iOS版本上持續執行而無需重大更新，展示了堅實的架構和前瞻性的開發。

使用的技術

Objective-C、MiLibris API v3/v4、iOS SDK

平台

適用於iPhone和iPad的通用應用程式
