SELECT - 你的社群

一個連接志同道合者的跨平台社交社群應用程式。我在Super Effective工作期間參與了SELECT的開發，為原生iOS版本和完整的React Native重寫做出了貢獻。

初始iOS版本 (v1.32-1.37)

效能最佳化

  • 將編譯時間從70分鐘減少到11分鐘
  • 使用CocoaPods模組化程式碼庫
  • 將架構重構為MVVM與依賴注入

DevOps

  • 使用Fastlane與Jenkins配置持續交付環境
  • 簡化部署流程

React Native重寫 (v1.38+)

核心功能

  • WebSocket通訊協定：定義並實現用戶端/伺服器協定
  • 多好友選擇器(MFS)：智慧聯絡人搜尋和建議系統
  • 自訂介紹畫面：深度連結啟動的自訂啟動畫面
  • 原生圖像選擇器：構建帶有React Native橋接的Objective-C模組以獲得最佳效能

UI實現

  • 負責使用Flexbox的大多數畫面佈局
  • iOS和Android上流暢、響應式的設計

使用的技術

React Native、Swift 3.0、Objective-C、WebSocket、Flexbox、Fastlane、Jenkins

平台

在iOS和Android上都可用
