SELECT - Your Community

A cross-platform social community application that connects people with shared interests. I worked on SELECT during my time at Super Effective, contributing to both the native iOS version and the complete React Native rewrite.

Initial iOS Version (v1.32-1.37)

Performance Optimization

  • Reduced compilation time from 70 minutes to 11 minutes
  • Modularized codebase using CocoaPods
  • Refactored architecture to MVVM with Dependency Injection

DevOps

  • Configured Continuous Delivery environment using Fastlane with Jenkins
  • Streamlined deployment process

React Native Rewrite (v1.38+)

Core Features

  • WebSocket Communication Protocol: Defined and implemented client/server protocol
  • Multi Friend Selector (MFS): Intelligent contact search and suggestion system
  • Custom Intro Screens: Deep-link activated custom launch screens
  • Native Image Picker: Built Objective-C module with React Native bridge for optimal performance

UI Implementation

  • Responsible for most screen layouts using Flexbox
  • Smooth, responsive design across iOS and Android

Technologies Used

React Native, Swift 3.0, Objective-C, WebSocket, Flexbox, Fastlane, Jenkins

Platform

Available on both iOS and Android
