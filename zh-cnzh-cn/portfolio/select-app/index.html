SELECT - 你的社区

一个连接志同道合者的跨平台社交社区应用。我在Super Effective工作期间参与了SELECT的开发，为原生iOS版本和完整的React Native重写做出了贡献。

初始iOS版本 (v1.32-1.37)

性能优化

  • 将编译时间从70分钟减少到11分钟
  • 使用CocoaPods模块化代码库
  • 将架构重构为MVVM与依赖注入

DevOps

  • 使用Fastlane与Jenkins配置持续交付环境
  • 简化部署流程

React Native重写 (v1.38+)

核心功能

  • WebSocket通信协议：定义并实现客户端/服务器协议
  • 多好友选择器(MFS)：智能联系人搜索和建议系统
  • 自定义介绍屏幕：深度链接激活的自定义启动屏幕
  • 原生图像选择器：构建带有React Native桥接的Objective-C模块以获得最佳性能

UI实现

  • 负责使用Flexbox的大多数屏幕布局
  • iOS和Android上流畅、响应式的设计

使用的技术

React Native、Swift 3.0、Objective-C、WebSocket、Flexbox、Fastlane、Jenkins

平台

在iOS和Android上都可用
