SELECT - Votre Communauté

Une application communautaire sociale multiplateforme qui connecte les personnes partageant les mêmes intérêts. J'ai travaillé sur SELECT pendant mon passage chez Super Effective, contribuant à la fois à la version iOS native et à la réécriture complète en React Native.

Version iOS Initiale (v1.32-1.37)

Optimisation des Performances

  • Réduit le temps de compilation de 70 minutes à 11 minutes
  • Modularisé la base de code en utilisant CocoaPods
  • Refactorisé l'architecture vers MVVM avec Injection de Dépendances

DevOps

  • Configuré l'environnement de Livraison Continue en utilisant Fastlane avec Jenkins
  • Rationalisé le processus de déploiement

Réécriture React Native (v1.38+)

Fonctionnalités Principales

  • Protocole de Communication WebSocket : Défini et implémenté le protocole client/serveur
  • Sélecteur Multi Amis (MFS) : Système intelligent de recherche et suggestion de contacts
  • Écrans d'Intro Personnalisés : Écrans de lancement personnalisés activés par deep-link
  • Sélecteur d'Images Natif : Module Objective-C construit avec pont React Native pour des performances optimales

Implémentation UI

  • Responsable de la plupart des layouts d'écran utilisant Flexbox
  • Design fluide et réactif sur iOS et Android

Technologies Utilisées

React Native, Swift 3.0, Objective-C, WebSocket, Flexbox, Fastlane, Jenkins

Plateforme

Disponible sur iOS et Android
