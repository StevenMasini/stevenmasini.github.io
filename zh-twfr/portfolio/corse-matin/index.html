Corse Matin per noï

Un lecteur de journal numérique pour l'agence de presse Corse Matin, offrant une visualisation PDF enrichie des publications quotidiennes et hebdomadaires.

Fonctionnalités Clés

  • Intégration API MiLibris : Intégré les versions v3 et v4 de l'API MiLibris pour la visualisation PDF enrichie
  • Version iPad : Conçu et développé la version optimisée pour iPad
  • Compatibilité à Long Terme : Construit pour iOS 4.3+, toujours fonctionnel sur iOS 11 sans mises à jour majeures

Réalisation Technique

Créé une application robuste et pérenne qui a continué à fonctionner sur plusieurs versions d'iOS sans nécessiter de mises à jour significatives, démontrant une architecture solide et un développement avant-gardiste.

Technologies Utilisées

Objective-C, MiLibris API v3/v4, iOS SDK

Plateforme

Application universelle pour iPhone et iPad
