Corse Matin per noï

Corse Matin报社的数字报纸阅读器，提供日报和周报的增强PDF查看功能。

主要功能

  • MiLibris API集成：集成MiLibris API的v3和v4版本以实现增强PDF查看
  • iPad版本：设计并开发了iPad优化版本
  • 长期兼容性：为iOS 4.3+构建，在iOS 11上仍能运行而无需重大更新

技术成就

创建了一个强健、经得起未来考验的应用程序，在多个iOS版本上持续运行而无需重大更新，展示了坚实的架构和前瞻性的开发。

使用的技术

Objective-C、MiLibris API v3/v4、iOS SDK

平台

适用于iPhone和iPad的通用应用
